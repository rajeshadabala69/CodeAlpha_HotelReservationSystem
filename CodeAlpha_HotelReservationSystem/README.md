# Hotel Reservation System

## Description
A Java-based hotel reservation system developed for CodeAlpha Internship.

## Features
- View available rooms
- Book rooms
- Cancel bookings
- Exit system

## Technologies Used
- Java
- Arrays
- OOP Concepts

## How to Run
javac HotelReservationSystem.java
java HotelReservationSystem
